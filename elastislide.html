Elastislide

Elastislide is a Responsive Image Carousel which can be used in standard (responsive) web pages and with the Layout Grid.

This extension is based on the Elastislide jQuery plugin by Mary Lou:
http://tympanus.net/codrops/2011/09/12/elastislide-responsive-carousel/
