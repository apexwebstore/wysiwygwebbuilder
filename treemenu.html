TreeMenu - Version 1.0

The TreeMenu extension displays a list of links in a tree-like format.
Levels can be expanded/collapsed by the user with the ‘+’ and ‘-‘ icons. As a designer you can set the inital state (expanded/collapsed) of the tree items.

The TreeMenu was created using jQuery.
