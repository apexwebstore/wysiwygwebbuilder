jQuery Accordion Widget (DEPRECATED)

The Accordion extension implements the jQuery Accordion Widget so it can be easily used in WYSIWYG Web Builder.

To edit the content of the panels, this extension uses the HTML TextBox extension. This gives you full WYSIWYG control over the content of the panels.

Key Features
• Add a cool accordion-like widget to your page without the need to manually edit HTML code.
• All required jquery code will be automatically generated for you!
• Edit the content of the panels using the feature rich HTML editor (the HTML TextBox extension).
• Panels can be activated on mouse click or mouse over.
• Slide animation for hiding and showing Accordion panels.
• Supports ThemeRoller themes, so you can easily modify the appearance of the Accordion Widget.
