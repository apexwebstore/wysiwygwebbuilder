News Writer - Version 1.3

Did you ever want to be able to edit parts of your website online?
Or have you designed websites for customers who want to be able to add/edit news items, agenda's or activities themselves?

If the answer is 'yes', then this News Writer extension might be the perfect solution for you. News Writer is a very simple online Content Management System (CMS) which allows you to manage a specific area of your webpage online.

Features:
• Manage news and other (HTML) content online.
• Items will be stored in a file on the server, no database required!
• Password protected, only the administrator can add/edit/remove items.
• Items can be re-arranged.
• Choose from a Text only editor or a fully featured online HTML editor!
• Upload images!
