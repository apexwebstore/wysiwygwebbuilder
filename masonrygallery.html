Masonry Gallery (PAID)


The Masonry Gallery extension is a grid based gallery which adapts to the screen size dynamically. The gallery supports images with different sizes and aspect ratios. Basically, Masonry optimizes the use of space inside the web page by reducing any unnecessary gaps. This extension uses David DeSandroâ€™s Masonry jQuery plug-in: http://masonry.desandro.com

Masonry Gallery can be used in responsive layouts (with breakpoints) or inside a layout grid for full width/flexible layouts.
Breakpoints can be added to adjust the column count for different screen sizes. Clicking on one of the images will trigger a lightbox slideshow.
Another cool feature is the ability to control the position and size of the caption for each image individually. This make it possible to position each caption in a way that works best for each image.
