Password Protect - Version 1.0

This extension adds very simple password protection to your webpage.
Simply drop the object to your page, set the password and it will prompt the visitor of the page for a password. This solution isn't the most secure way to protect your page, and experienced users probably will not have to much trouble finding a way around this.
Note that users must have Javascript enabled or the page will not be protected.
For a more reliable protetion methid please visit:
https://www.wysiwygwebbuilder.tk/password_protect.html

Properties:
Password: the password for the protected page.
Wrong password URL: if the password is wrong or you press Cancel you will be redirected to this URL.
Password Prompt: the password prompt text.
