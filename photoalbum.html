Photo Album - Version 1.2

The Photo Album extension introduces yet another cool way to display pictures on your web site.

Basically this extension works the same way as the standard Photo gallery object in WYSIWYG Web Builder, but the output is completely different. Rather than displaying all pictures on 1 page, you can display images over multiple pages. The thumbnail navigation option provides a filmstrip-style menu to navigate through the images.

Features:
• Create great looking photo albums with only a few mouse clicks.
• Customize the appearance (colors, fonts) of the photo album, without having to edit HTML code!
• Add a title and description for each photo.
• Choose the number of rows and columns and the thumbnail width and height for the index page(s).
• Choose whether navigation controls appear above or below the large photo on photo pages.
• Filmstrip-style navigation.
