Online Slide Show - Version 1.1

The Online Slide Show extension is an online version of the standard Slide Show object in WYSIWYG Web Builder. The main difference is that it can display images from a folder on the server without the need to republish the page..

Features:
• Loads images from a folder on the server.
• No database required.
• Supports many effects/animations: fade, slide, ken burns, conveyer belt etc.
• Built-in support for WWB8 lightbox add-ons!
