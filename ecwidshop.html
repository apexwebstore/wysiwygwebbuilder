Ecwid Web Shop Extensions

This is a set of extensions for integrating the easy but powerful Ecwid eCommerce platform: http://www.ecwid.com/
Ecwid is an online e-commerce platform that allows you to add shopping cart functionality to any current site and also get a mobile and Facebook store in one package. 

The following extensions are included:

MiniCart
This widget contains the Shopping Bag; when viewing products you can Drag & Drop them to the shopping basket.

Vertical Categories
Display a list of categories using a vertical layout

Horizontal Categories
Display a list of categories using tabs

Language
Change the language of your store.

Product Browser
This is the main container and it is mandatory; this will contain your products, categories, search results, the checkout options etc.

Search Box
Search for products
