jQuery GoUp!

The jQuery GoUp! extension is a simple jQuery plugin that let users of your web pages go back directly to the top.
This extension is a wrapper for the jQuery GoUp! plugin by Daniele Lenares. http://dnlnrs.github.io/jquery-goup/
