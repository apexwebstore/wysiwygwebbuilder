File Publisher Plus (DEPRECATED)

This extension is an advanced version of the standard File Publisher object in WYSIWYG Web Builder. The main difference is that now you can upload complete folder structures.

The File Publisher Plus extension automatically uploads the specified files to your web site and creates the folders once you publish this page.

This object can be useful if you want to include extra files which are part of the page (like scripts or multimedia files).
