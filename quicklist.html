QuickList - Version 1.0

The Quick List extension is an object that populates a combobox/listbox with the contents of a text file.
This allows you for example to quickly add a Country selection box to your page.
1. Add the object to your page
2. Select a text file which contains the list (using the Browse button).
3. Set some other parameters of the control (liek the Default to be displayed)
4. Done!
Besides countries you can of course also create other lists like states, colors, cities and much more.
The file with items must have two items per line, seperated by a semi colon:
Item Text;Item Value

Most other parameters of this object are the same as the regular ComboBox object in Web Builder.
