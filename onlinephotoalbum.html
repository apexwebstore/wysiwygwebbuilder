Online Photo Album - Version 1.1

Online Photo Album is an easy and feature-rich online photo album solution for WYSIWYG Web Builder. It can display images from a folder on the server without the need to republish the page.

Features:
• Loads images from a folder on the server.
• No database required.
• Supports pagination and ‘Next/Previous’ navigation.
• Many customization options.
• Supports images border and CSS3 box shadow.
• Built-in support for WWB8 lightbox add-ons!

Requirements:
WYSIWYG Web Builder 11.6 or higher
