Shiny Button Extension

Requirements:
WYSIWYG Web Builder 9.0 or higher
Shiny Button

This extension simulates a ‘shiny’ glass button and uses pure CSS3. It has several customization options for the appearance of the button. It’s also possible to set the output format to a ‘png’ image in case you want to target older browsers that do not support CSS3 (IE9 and older).
