PHP File Listing - Version 1.2

The PHP File Listing extension generates a listing of the contents of a specified directory on your web site. Let’s say for example you have a directory on your web site with MP3 files then this extension will automatically scan this folder and display the contents on your web page as links, so users can download these files.
