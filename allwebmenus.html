AllWebMenus - Version 1.0

The AllWebMenus extension provides an easy way to integrate Likno's AllwebMenus into your pages and have access to hundreds of cool menu designs.

Likno's AllWebMenus is a DHTML menu/JavaScript menu builder to create any kind of popup or drop-down menu for your web navigation needs, without programming experience required. Use plenty of stylish themes to easily build SEO-friendly web menus, CSS menus, Sliding menus, Floating menus, Mega drop-down menus, AJAX menus etc. that work alike in all DHTML browsers (IE, Firefox, Mozilla, Safari, Chrome, Opera etc.) and all popular platforms (Win, Mac, Unix). Create pop-up, slide or drop-down menus with static or animated images, borders, effects, etc.

Visit the Likno website for more details: http://www.likno.com/

Now you can integrate these powerful menus in your WYSIWYG Web Builder web site with only a few mouse clicks!
