CSV CRUD Extension


Requirements:
WYSIWYG Web Builder 15.1 or higher
CSV Create, Read, Update, Delete

CSV CRUD is a CSV editor for WYSIWYG Web Builder that will allow you to create, read, update and delete records of a CSV file on your website. The acronym CRUD stands for Create, Read, Update and Delete..

Features
• Create, read, update and delete csv records
• built-in search
• sort columns
• pagination (number of items per page)
• bootstrap 4 look and feel
• specify required input fields
• specify maximum length of input fields
• configure which fields to display in the listview
• input types: editbox, textarea, combobox (with options) and checkbox
• configurable text labels (language)
• many style options: colors, padding, font, bordered table etc.
