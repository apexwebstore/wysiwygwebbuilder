Simple Viewer (DEPRECATED)

This extension is a wrapper for the Simple Viewer Image viewer by Felix Turner.
http://www.simpleviewer.net/simpleviewer

Simple Viewer is a cool way to add a professional flash image viewer to your web site.
This extension for WYSIWYG Web Builder automatically generates all the necessary
HTML code, data files and thumbnails for you!

Features:
• Intelligent image pre-loading.
• Intuitive image navigation
• Customizable interface - Set text color, frame color and width, navigation position.
• Cross platform - Windows/Macintosh/Linux/iPhone/iPad and Android.
• Keyboard Navigation (Cursor keys, Home, End, Page Up/Down)
• Mousewheel navigation

Note:
SimpleViewer-Pro is also available for purchase. SimpleViewer-Pro does not include the download link.
http://www.simpleviewer.net/simpleviewer/pro
