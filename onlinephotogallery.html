Online Photo Gallery - Version 1.2

Online Photo Gallery is an easy and feature-rich online (database driven) photo gallery solution for WYSIWYG Web Builder to manage photos online.

Features:
• Manage photos online (no need to republish the page when you add new photos).
• Uses MySQL to store information about the photos (title, keywords etc).
• Upload images via admin.
• Automatically creates thumbnail images for uploaded images.
• Supports images border and CSS3 box shadow.
• Built-in support WWB8 lightbox add-ons!
• Optional RSS feed
• Photos are searchable based on keywords.
• Supports localization.
