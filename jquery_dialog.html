jQuery Dialog Widget (DEPRECATED)

The Dialog extension implements the jQuery Dialog Widget so it can be easily used in WYSIWYG Web Builder. A dialog is a floating window that contains a title bar and a content area. The dialog window can be moved, resized and closed with the 'x' icon by default..

To edit the content of the panels, this extension uses the HTML TextBox extension. This gives you full WYSIWYG control over the content of the panels.

Key Features
• Add cool popup windows to your pages without the need to manually edit HTML code.
• All required jquery code will be automatically generated for you!
• Edit the content of the dialog using the feature rich HTML editor (the HTML TextBox extension).
• Modal dialogs.
• Draggable.
• Resizable.
• Built-in support for many show/hide effects!
• Supports ThemeRoller themes, so you can easily modify the appearance of the Dialog Widget.
