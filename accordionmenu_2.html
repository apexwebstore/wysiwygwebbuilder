Accordion Menu

A popular implementation of the jQuery Accordion object (available in the WYSIWYG Web Builder toolbox) is as a navigation menu.
To make it even easier to create such a menu, we've created the Accordion Menu. It's a simple extension of the standard accordion widget.  All you will have to do is specify the headers and links, configure the style and the menu is ready to use!

The Accordion Menu was created using jQuery and supports ThemeRoller themes.
