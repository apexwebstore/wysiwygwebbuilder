News Writer Pro - Version 1.2

This is an advanced version of the News Writer extension which is separately available: https://www.wysiwygwebbuilder.tk/newswriter.html

Did you ever want to be able to edit parts of your website online? Or have you designed websites for customers who want to be able to add/edit news items, agenda's or activities themselves? If the answer is 'yes', then this News Writer extension might be the perfect solution for you. News Writer is a very simple online Content Management System (CMS) which allows you to manage a specific area of your webpage online. You can also use it as simple blog or as (online editable) data source for WYSIWYG Web Builder’s News Feed Ticker.

Features:
• Manage news and other (HTML) content online.
• Password protected, only the administrator can add/edit/remove items.
• Items can be re-arranged.
• Upload images.

Pro features:
• Items will be stored in a MYSQL server. Only one database/table is needed for multiple news writers.
• Supports localization.
• Supports several popular editors: TinyMCE, CKEdit, CLEdit and the WWB Editor.
• Built-in support for RSS feed so you can use it as data source for WYSIWYG Web Builder’s News Feed Ticker or other RSS viewers.
• SEO friendly (content will be embedded in the same page).
