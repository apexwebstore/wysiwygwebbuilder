Justified Gallery
The Justified Gallery extension is a grid based gallery which adapts to the screen size dynamically. The gallery supports images with different sizes and aspect ratios, the images will automatically be re-arranged based on the available space.
This extension uses Miro Manninoâ€™s Justified Gallery jQuery plug-in: http://miromannino.github.io/Justified-Gallery/

Justified Gallery can be used in responsive layouts (with breakpoints) or inside a layout grid for full width/flexible layouts.
Clicking on one of the images will trigger one of the built-in lightbox slideshows.
