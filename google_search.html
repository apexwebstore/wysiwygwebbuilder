Google Search (DEPRECATED)

With the Google Search extension, you can easily incorporate a Google Web Search module on your Web site and allow site visitors to search the World Wide Web directly from your site.
Use Google Free search the same way you use Google.com and you'll get exactly the same results.

This extension supports 3 Google search services:
Google Free Web Search, searches the entire World Wide Web.
Google Free SafeSearch, searches the entire World Wide Web while filtering adult content.
Google Free Web Search with site search, searches specific domains.

When you select 'Google Free Web Search with site search' you can also customize the results page to match the style of your website.

For more details about these free Google services please visit:
http://www.google.com/searchcode.html
http://www.google.com/faq_freewebsearch.html
